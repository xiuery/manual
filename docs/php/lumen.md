#### 环境要求
- PHP >= 7.2
- OpenSSL PHP 拓展
- PDO PHP 拓展
- Mbstring PHP 拓展

#### Installation
```

composer global require "laravel/lumen-installer"

# 创建lumen项目
lumen new xs-auth
```
